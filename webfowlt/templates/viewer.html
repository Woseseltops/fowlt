    Error tolerance

    For many errors, Fowlt is not entirely sure whether it is a real error or not. This number represents how certain Fowlt should be before it flags a word as an error. If you set this to 0 Fowlt will only show errors it is very certain about. When you set this to 100 Fowlt will flag everything that can possibly be an error.
